Bienvenido a mi pagina HTML


HTML es un lenguaje con el que se escriben las páginas web pueden ser visitadas por el usuario mediante un tipo de aplicación llamada navegador.

Este lenguaje nos permite alutinar textos, sonidos e imágenes y combinarlos a nuestro gusto.

el HTML nos permite la introducción de referencias a otras páginas por medio de los enlaces hipertexto
